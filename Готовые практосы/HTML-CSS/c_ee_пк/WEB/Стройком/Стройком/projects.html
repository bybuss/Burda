Какие дома мы строим?

Мы строим дома по различным технологиям. Обычно заказчик выбирает каким будет его будущий дом, но в некоторых случаях тип грунта на земельном участке "диктует" свои условия. На подвижных почвах целесообразно возводить только монолитные дома, именно такая технология убережет ваш дом от трещин. Тип грунта на большинстве участков в Дагестане позволяет построить дом по любой из перечисленных ниже технологий.
